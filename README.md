# WelcomeToJPG

WelcomeToJPG is a macOS desktop app for converting multiple images to JPG in one place.

## Features

- Convert `HEIC`, `PNG`, `JPEG`, and other supported image formats to JPG
- Add files with drag and drop or a file picker
- Preview selected images directly inside the app
- Choose JPG compression quality before converting
- Save converted files to a selected folder with automatic filename conflict handling
- Flatten transparent images onto a white background for JPG output
